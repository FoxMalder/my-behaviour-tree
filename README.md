# MyBehaviourThree

My implementation of behaviour three for game AI.

## Development

First install dependencies:

```sh
npm install
```

To run in hot module reloading mode:

```sh
npm start
```

To create a production build:

```sh
npm run build-prod
```
Running

```sh
node dist/bundle.js
```

## Built With

Javascript, [parcel](https://github.com/parcel-bundler/parcel)
